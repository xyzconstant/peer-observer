IP-ranges and ASNs

Shows the top IP-ranges and ASNs for currently connected peers and in addrman tables.

Current peers

Shows the top IP addresses and ASNs for the currently connected peers. Data comes from the getpeerinfo RPC and is updated every 10s.

Addrman tables

Shows the top IP addresses and ASNs (if ASMap is enabled) for the address manager (addrman) tables. Uses data from the getrawaddrman RPC and is updated every 120s.
